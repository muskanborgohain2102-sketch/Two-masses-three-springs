# Structural Dynamics Simulation of a Coupled Two Degree of Freedom System
The code hereby displayed is a python-based numerical solver for a coupled two Degree of Freedom (DoF) vibration system using NumPy, SciPy and Matplotlib. It has been developed to –
•	Solve coupled Ordinary Differential Equations (ODEs) governing spring-mass dynamic response under varying initial conditions.
•	Implement damped and undamped dynamic response models by solving coupled second-order ODEs using numerical integration techniques.
•	Generate displacement-time response plot, and analyse damped resonance characteristics relevant to aerospace structural vibration analysis.
